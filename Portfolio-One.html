JavaScript Calculator

One of my first projects at the Center for Advanced Research and Technology, was the JavaScript calculator. This project took place at the beginning of the school year when we were introduced to the coding language JavaScript. The software that I used to code the JavaScript calculator is Visual Studio. With visual studio, I was able to write functions that would be able to add, subtract, multiply, and divide. The resources that were given to me was, lessons about JavaScript from my learning instructor Mr. Fast, and a book called Eloquent JavaScript. In the book Eloquent JavaScript, I learned how functions work, and how you’re supposed to write them. The coding language JavaScript was difficult, and I had to restart my functions everytime it didn’t work the way that I wanted it to work. I struggled with this project, however I kept using the resources that were given to me, and I also communicated with some of my classmates, which resulted in me completing the JavaScript Calculator. With the communication that I had with my instructor and my classmates, it aided me in completing this project.
