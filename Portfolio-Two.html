Interactive Story

A project in which I was able to work together with a partner, was the Interactive Story project. In this project, we used the short stories that we wrote in english, to create a JavaScript interactive story. My partner and I worked on creating functions such as, Function makeButton, Function buildStory, and Function showStory. Before we created these functions, we communicated with each other and created a flowchart where we listed the different types of options and scenarios that you could do if you chose a certain path in the story. In the early stages of this project, there were two flowcharts, the Story Flowchart and the Code Flowchart, these flowcharts helped my partner and I in the completion of our project, because it made things very simple for us. Some of the resources we used were websites such as LucidChart, Google Keep, and W3 Schools. The coding language that my partner and I used was JavaScript. One of our most difficult tasks was to create a fully functional button, we kept failing, however on our last try it finally worked. The software my partner and I used was Visual studio.
