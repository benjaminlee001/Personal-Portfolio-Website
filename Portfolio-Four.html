CSS Zen Garden Challenge

Cascade Styling Sheets, that is one of the last coding languages that I was taught, assignment where we exercised our learning of CSS was the Zen Garden Challenge. In the Zen Garden Challenge, we were tasked in using CSS styling on the Zen Garden HTML template. Essentially, I used my knowledge of CSS by centering some parts of the text, changing the fonts, adding borders to the text, and resizing the text. I also used some of the knowledge I gained from the flexbox lesson that my instructor taught us, to create a type of flexbox layout. The software that I used was Visual Studio, after I used my knowledge of CSS on the Zen Garden HTML template, I uploaded the file to GitHub, where I stored my code.
