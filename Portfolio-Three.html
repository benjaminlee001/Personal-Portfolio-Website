HTML Images

An assignment where I used the coding language HTML, was the HTML Images assignment. In this assignment, I was tasked to choose either Fiji or Norway, so I decided to choose Norway because of the beauty of the country. In this assignment, I was tasked to pick a few images of Norway, which I then described the images and listed facts about the country, using the coding language HTML. Out of all of the projects and assignments I have done, this one was my favorite, because we were given the opportunity to use the coding language HTML in a fun way of describing a wonderful country. In this assignment, the resources that I used was the lesson that my instructor taught about HTML. The software that I used to complete this assignment was Visual Studio, in visual studio I added images in and described them using HTML.
